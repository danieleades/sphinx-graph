Welcome to sphinx-graph's documentation!
========================================

Sphinx-Graph is a sphinx extension for plain-text requirements management.

It provides the :doc:`vertex </src/vertex>` directive for defining relationships between arbitrary elements in a Sphinx document.

The :doc:`vertex-table </src/table>` directive can be used for generating summary tables of vertices.
